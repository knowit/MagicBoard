## Using the module

To use this module, add the following configuration block to the modules array in the `config/config.js` file:
```js
All config values are default and only needs to be added if changed.
{
    module: 'MMM-Boards',
    config: {
        updateInterval: 1000 * 60
        fadeSpeed: 1000 * 2
    }
}
```
